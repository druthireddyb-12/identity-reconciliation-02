# Identity Reconciliation

Simple Node.js backend implementing the Task 1 "Identity Reconciliation" API.

Files:
- index.js — main Express app and reconciliation logic
- test/identify.test.js — test suite

Quick start

1. Install dependencies

```bash
npm install
```

2. Run tests

```bash
npm test
```

3. Run server (optional)

```bash
node index.js
```
